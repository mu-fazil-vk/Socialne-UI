## Test the Socialne UI

1. Open your terminal or command prompt.
2. Navigate to the directory where you want to clone the repo.
3. Run the following command:

```bash
  git clone https://github.com/mu-fazil-vk/Socialne-UI.git
  cd Socialne-UI
  flutter run
```
